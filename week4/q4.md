# Security Policy

## A. Password Policy
- **Minimum Length:** All passwords must be at least 12 characters.
- **Complexity:** Must include uppercase, lowercase, numbers, and symbols (e.g., @, #).
- **Rotation:** Passwords must be changed every 90 days.
- **Storage:** Use of a centralized password manager (e.g., Bitwarden) is mandatory; writing passwords on paper is prohibited.

## B. Data Backup Policy
- **Frequency:** Incremental backups daily at 6:00 PM; Full system backups every Sunday.
- **Location:** Use the 3-2-1 Rule: 3 copies of data, 2 different media (Cloud + External Drive), 1 copy stored off-site.
- **Verification:** Backups must be tested for restoration once a month.

## C. Internet Usage Guidelines
- **Business Only:** Internet is primarily for work-related tasks. Accessing gambling or high-risk sites is blocked.
- **Downloads:** No third-party software or browser extensions may be installed without IT approval.
- **Hardware:** Personal USB drives are prohibited to prevent "Rubber Ducky" attacks or malware transfer.
